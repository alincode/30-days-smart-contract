# 錢包 (wallet)

錢包就是管理鑰匙的工具軟體

我們前一篇有提到，當新增一個交易時，使用者需要進行「交易確認」，

我們並不是直接跟網路協定溝通，而是透過友的善瀏覽器介面，例如 chrome、firefox、IE 來讀取跟發佈資料。

**常見功能**

* 新增帳戶
* 匯入帳戶
* 進行轉帳
* 查詢餘額
* 交易的確認
